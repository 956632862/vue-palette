# canvas-palette

## 简述 
这个项目主要是VUE版本的涂鸦板，目前只支持在PC使用，
只完善了基础的功能

## 已完成的功能
-[x] 涂鸦
-[x] 修改画笔颜色
-[x] 修改画笔大小
-[x] 上一步功能
-[x] 下一步功能
-[x] 切换背景图片


## 计划完成的功能

<input type="checkbox"> 支持拖拽图片 </input><br>
<input type="checkbox"> 支持缩放图片 </input><br>
<input type="checkbox"> 支持生成图片 </input><br>

yarn install
